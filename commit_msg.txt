Phase I (Opt suite): 6/7 bottleneck optimizations

1. FAISS/vectorized scans (Opt 1):
- Replaced brute-force O(N) Python dot-product loops with vectorized
  matrix multiply in _auto_expand_concepts and _learn_from_text
- Added _vectors_dirty guard to rebuild matrix before use

2. Parallel article fetching (Opt 2):
- Replaced sequential article fetch loop with ThreadPoolExecutor
  (max_workers=4), reducing 3 sequential fetches to parallel

3. Threshold-filter decoder vocab (Opt 5):
- Cerebellar n-gram and basal ganglia loops now pre-filter to
  tokens with logits > 0.01 via np.where (20-50 tokens vs 350)

4. Hashmap edge repetition penalty (Opt 6):
- Added _recent_traversal_map: Dict[Tuple[int,int], int] alongside
  _recent_traversals list — O(1) dict lookup replaces O(n*30) scan
- Map rebuilt from scratch after list pruning (n <= 30, negligible)

5. Lightweight pickle (Opt 7):
- Added __getstate__ on ConceptEdge excluding agent_weights,
  source_metadata, and parent_graph — cuts ~2-3 MB from 6 MB pickle
- parent_graph restored by ConceptGraph.__setstate__ as before

6. Duplicate _rebuild_raw_cache fix (Opt 4):
- Removed duplicate loop in module.py load_state_dict (was running
  the same _rebuild_raw_cache loop twice consecutively)
